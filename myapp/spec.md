# AI Agent Instruction: Project Specification
**IMPORTANT DIRECTIVE FOR AI AGENT:** Always read and reference this `spec.md` file before starting a new session or generating content for this project.

## Project Overview
**Project Name:** M.3 Student Portfolio Generator
**Objective:** To generate structured, high-quality portfolio content for 9th-grade (M.3) students applying for high school (M.4) using the customized 5W1H framework (based on the provided diagram).

## 5W1H Specification Framework

### 1. What (สร้างอะไร)
*   **Product:** แฟ้มสะสมผลงาน (Portfolio)
*   **Target Use Case:** สำหรับนักเรียนชั้น ม.3 เพื่อใช้ยื่นประกอบการสัมภาษณ์เข้าศึกษาต่อในระดับชั้น ม.4 (พร้อมรองรับการปรับแต่งตามสายการเรียน เช่น วิทย์-คณิต, ศิลป์-คำนวณ, ศิลป์-ภาษา)

### 2. Who (กลุ่มผู้ใช้และคนอ่าน)
*   **Target Audience:** คุณครูแนะแนว และคณะกรรมการสอบสัมภาษณ์ของโรงเรียนปลายทาง
*   **Context:** กรรมการมีเวลาจำกัดในการอ่าน ดังนั้นเนื้อหาต้องดึงดูด ทำให้เห็นความสามารถ ทัศนคติ และความมุ่งมั่นของนักเรียนได้อย่างรวดเร็ว

### 3. Why (ทำไมถึงสร้าง / แก้ปัญหาอะไร)
*   **Purpose:** เพื่อนำเสนอตัวตน จุดเด่น และผลงานที่ผ่านมาให้โดดเด่นและน่าประทับใจ
*   **Value Proposition:** เปลี่ยนจากการเล่าเรื่องปากเปล่ามาเป็นการนำเสนอด้วยหลักฐานเชิงประจักษ์ (รูปภาพ, เกียรติบัตร, ผลงาน) ที่มีการจัดระเบียบอย่างดี

### 4. Features (องค์ประกอบหลักที่ต้องมี)
*   ต้องมีโครงสร้างเนื้อหาครอบคลุม 5 ส่วนนี้:
    1. **Profile:** ประวัติส่วนตัว ข้อมูลการติดต่อ และคติประจำใจ
    2. **Education:** ประวัติการศึกษา และเกรดเฉลี่ย
    3. **Skills:** ทักษะทางวิชาการ ทักษะทางสังคม (Soft Skills) และความสามารถพิเศษ
    4. **Awards:** ผลงานและเกียรติบัตรที่โดดเด่นที่สุด (คัดมา 3-5 ชิ้น พร้อมคำอธิบายสั้นๆ)
    5. **Activities:** กิจกรรมเพื่อสังคมหรือกิจกรรมโรงเรียนที่เคยเข้าร่วม

### 5. Look (หน้าตา สไตล์ Layout)
*   **Style:** Clean & Minimal สะอาดตา สบายตา
*   **Layout:** จัดวางเป็นระเบียบ แบ่งสัดส่วนชัดเจน มีพื้นที่ว่าง (White space) ให้อ่านง่าย
*   **Color Scheme:** โทนสีเหมาะสมกับภาพลักษณ์และสายการเรียน
*   **Media:** มีการระบุตำแหน่งวางรูปภาพ (Placeholder) อย่างชัดเจน

### 6. How (วิธีการนำเสนอและข้อควรระวัง)
*   **Platform/Format:** จัดทำเป็นเอกสารขนาด A4 (เน้นการส่งออกเป็นไฟล์ PDF)
*   **Writing Style:** กระชับ ตรงประเด็น ใช้ Bullet points ในการอธิบายผลงานและกิจกรรม
*   **Constraints (สิ่งที่ห้ามมี):** ห้ามเขียนเรียงความยาวๆ (Wall of text) และห้ามใส่ข้อมูลที่ไม่เกี่ยวข้องกับการเรียนต่อ (เช่น กิจกรรมยามว่างที่ไม่มีผลต่อทักษะ)
